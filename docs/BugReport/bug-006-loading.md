BUG-006 – Loading desactualizado

ID: BUG-006
Severidad: Baja
Prioridad: Baja

Descripción

El loading animation utilizado en la web pertenece al estilo visual del congreso 2024, y no coincide con la identidad visual del evento actual.

Pasos para reproducir

Abrir la web.

Recargar la página.

Observar la animación de carga inicial.

Resultado esperado

El loading debe corresponder al diseño visual del congreso actual.

Resultado actual

Se muestra la animación del evento 2024.
