BUG-004 – Hover del menú no visible

ID: BUG-004
Severidad: Baja
Prioridad: Baja
Dispositivo: Desktop
Navegador: Chrome / Firefox

Descripción

En desktop el estado hover del menú de navegación es casi imperceptible, lo que puede afectar la experiencia del usuario al no identificar claramente el elemento interactivo.

Pasos para reproducir

Abrir la web en desktop.

Pasar el cursor sobre los elementos del menú principal.

Resultado esperado

El hover debe ser claramente visible para indicar interacción.

Resultado actual

El cambio visual es mínimo y difícil de percibir.
