BUG-003 – Texto cortado en sección "Nosotros" (Firefox)

ID: BUG-003
Severidad: Media
Prioridad: Media
Navegador: Firefox
Dispositivo: Tablet (768px)

Descripción

El mismo problema de visualización del texto en la sección Nosotros también ocurre en el navegador Firefox, lo que sugiere un posible problema de estilos CSS.

Pasos para reproducir

Abrir la web en Firefox.

Cambiar resolución a 768px.

Navegar a la sección Nosotros.

Resultado esperado

El texto debe visualizarse completo.

Resultado actual

El texto aparece cortado.
