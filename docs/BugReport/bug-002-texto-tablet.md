BUG-002 – Texto cortado en sección "Nosotros"

ID: BUG-002
Severidad: Media
Prioridad: Media
Dispositivo: Tablet (768px)
Navegador: Chrome

Descripción

En resolución tablet, el párrafo principal de la sección "Nosotros" aparece cortado y no permite visualizar correctamente todo el contenido.

Pasos para reproducir

Abrir la web del congreso.

Cambiar la resolución a 768px.

Navegar hasta la sección Nosotros.

Resultado esperado

El texto debe visualizarse completo y correctamente alineado.

Resultado actual

El párrafo aparece cortado y no se muestra completamente.
